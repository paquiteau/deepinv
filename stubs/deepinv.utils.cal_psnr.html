cal_psnrÔÉÅ

deepinv.utils.cal_psnr(a: Tensor, b: Tensor, max_pixel: float = 1.0, normalize: bool = False, mean_batch: bool = True, to_numpy: bool = True)[source]ÔÉÅ

Computes the peak signal-to-noise ratio (PSNR)

If the tensors have size (N, C, H, W), then the PSNR is computed as

\[\text{PSNR} = \frac{20}{N} \log_{10} \frac{\text{MAX}_I}{\sqrt{\|a- b\|^2_2 / (CHW) }}\]

where \(\text{MAX}_I\) is the maximum possible pixel value of the image (e.g. 1.0 for a normalized image), and \(a\) and \(b\) are the estimate and reference images.

Parameters:

  • a (torch.Tensor) ‚Äì tensor estimate

  • b (torch.Tensor) ‚Äì tensor reference

  • max_pixel (float) ‚Äì maximum pixel value

  • normalize (bool) ‚Äì if True, a is normalized to have the same norm as b.

  • mean_batch (bool) ‚Äì if True, the PSNR is averaged over the batch dimension.

  • to_numpy (bool) ‚Äì if True, the output is converted to a numpy array.
